## Try it with docker

```bash
docker run -w /root -it --rm alpine:edge sh -uelic '
  apk add git neovim ripgrep fd fzf sqlite-dev go npm alpine-sdk --update
  git clone https://github.com/treramey/nvim.git ~/.config/nvim
  cd ~/.config/nvim
  nvim
'
```
